# uk-crypto-tax-report

## Overview

This project is a personal crypto tax reporting tool built for my own use.

I created it because most commercial crypto tax platforms require users to provide API keys or upload full trading history to third-party services. From a security and privacy perspective, I prefer to process my data locally and retain full control over my exchange credentials and transaction history.

This tool calculates UK crypto capital gains based on my own trading patterns and workflow.

## Important Disclaimer

- This project was developed for personal use.
- It reflects my own trading setup and assumptions.
- It may not work correctly for other trading strategies, exchanges, or edge cases.
- It is not financial advice.
- Use at your own risk.
